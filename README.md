CommunityJSData
===============

The actual data for [CommunityJS.org](http://CommunityJS.org)

## Quick start
Only edit rawfile

or

Clone the repo

```
$ git clone git://github.com/CommunityJS/CommunityJSData.git
```

Go to folder

```
cd CommunityJSData
```

Edit rawdata file

Check your browser

[Make a pull request](https://help.github.com/articles/using-pull-requests)
